## Development

### Create (and activate) a virtual environment

```shell
python3 -m venv .venv
source .venv/bin/activate
```

### Install the project

```shell
pip install -r requirements.txt
pip install -e .
```

### Test

```shell
python -m pytest
```

aoc --years 2019 2018
